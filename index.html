このディスクについて

ご利用になるには、左のメニューからご希望の項目をクリックしてください。

Webブラウザの設定

JavaScriptを使用していますので、ご覧になるときは設定を有効にしてください。
詳しくは、「JavaScriptの設定」を参照してください。

画像を見る

診断画像ディスクの画像を見るには、以下のいずれかの項目をクリックしてください。

Webブラウザで見る

Webブラウザを使って患者データの概略を手早く参照できます。

専用ビューワで見る

専用のビューワを起動して診断画像を閲覧します。
長さや角度を測ったり、トーンを変更することもできます。
操作については専用ビューワのヘルプを参照してください。

動作環境

OS ：Windows XP SP3, Windows Vista, Windows 7, Windows 8.1, Windows 10
Webブラウザ ：Internet Explorer 5.5以降
メモリ ：2GB以上

OSのバージョン確認方法

以下の方法で確認してください。

  1. デスクトップの「マイ コンピュータ」を右クリックします。
    メニューが表示されます。
  2. 「プロパティ」を選択します。
    「システムのプロパティ」が表示されます。

「全般」タブの「システム」にバージョンやサービスパックが表示されています。

JavaScriptの設定

この診断画像ディスクは、JavaScriptを使用しています。
ご覧になるときは、Webブラウザの設定でJavaScriptを有効にしてください。（下記の操作手順は、Internet Explorerの例です。）

  1. InternetExplorerの「ツール」メニューの「インターネットオプション」を選択します。
  2. 「セキュリティ」タブをクリックし、「レベルのカスタマイズ」をクリックします。
  3. 「スクリプト」項目内の「アクティブスクリプト」の「有効にする」をクリックして選択し、「OK」をクリックします。
