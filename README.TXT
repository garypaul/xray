◆ディスク上のデータ構成に関して
 　ルートディレクトリ内容
　 　・DICOMDIRファイル
              -DICOMDIRが参照するすべてのDICOMインスタンスはディスク上に存在します。
　 　・DICOMフォルダ
              -ディスク上のすべてのDICOMファイルがあります。
　 　・INDEX.HTMファイル
              -画像をWebブラウザで参照するときのトップページです。
              -ディスクによっては存在しないことがあります。
　 　・IHE_PDIフォルダ
              -画像をWebブラウザで参照するときに必要なデータが置かれています。
              -ディスクによっては存在しないことがあります。
　 　・README.TXTファイル
              -このドキュメントファイルです。
　 　・REPORTフォルダ
              -レポートPDFをWebブラウザで参照するときに必要なデータが置かれています。
              -ディスクによっては存在しないことがあります。
　 　・VIEWERフォルダ
              -ビューワソフトウェアが置かれています。
　 　・VINFOフォルダ
              -ビューワソフトウェアに関連するデータが置かれています。
　 　・ATTACHEDフォルダ
              -追加ファイルが置かれています。
              -ディスクによっては存在しないことがあります。
