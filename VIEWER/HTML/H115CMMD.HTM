コマンド一覧

青字のタイトルをクリックすると、下位の項目の表示／非表示が切り替わります。

「ファイル」メニュー
  • 「すべての画像をファイル出力」
    表示されているすべての画像をファイルに書き出します。
  • 「選択画像をファイル出力」
    選択画像をファイルに書き出します。
  • 「カレント画像をファイル出力」
    選択されたカレント画像をファイルに書き出します。画像を右クリックして表示されるメニューから「この画像をファイル出力」を選択しても同じことができます。
  • 「閉じる」
    ビューワの画面を閉じます。閉じるボタン（）をクリックしても同じことができます。

「編集」メニュー
  • 「画像表示リセット」
    画像の明るさ、コントラスト、回転などの表示状態を、最初に開いた時点に戻します。
    → 明るさ・コントラスト（トーン）を調整する
  • 「コピー」
    領域指定されている範囲の画像データをクリップボードにコピーします。
  • 「切り抜き」
    領域指定されている範囲のみを残して画像を切り取ります。
    → 領域を選択する／切り抜く
  • 「画像全体を指定」
    選択されている画像の全体を対象領域にします。
    → 領域を選択する／切り抜く
  • 「対象領域の解除」
    指定している対象領域を解除します。
    → 領域を選択する／切り抜く
  • 「すべてのビューを選択」
    ビューワで開いているすべての画像を選択画像にします。
    → 操作する画像を選ぶ／複数の画像を同時に操作する（連動）
  • 「すべてのビューの選択を解除」
    ビューワで開いているすべての画像の選択を解除します。
    → 操作する画像を選ぶ／複数の画像を同時に操作する（連動）
  • 「右90度回転」
    ビューワで開いている画像を時計回りに90度回転させます。ツールバーの「90°右回転」ボタンと同じ機能です。
  • 「左90度回転」
    ビューワで開いている画像を反時計回りに90度回転させます。ツールバーの「90°左回転」ボタンと同じ機能です。
  • 「水平反転」
    ビューワで開いている画像を横に反転させます。ツールバーの「左右反転」ボタンと同じ機能です。
  • 「垂直反転」
    ビューワで開いている画像を縦に反転させます。ツールバーの「上下反転」ボタンと同じ機能です。
  • 「画像のソート」
    画像の表示順序を変更します。
    → ビューワの表示設定

「表示」メニュー
  • 「全体表示」
    画像全体がビューエリア全体に入るように自動的に調整し画像を表示します。（表示 される画像のサイズはウィンドウの大きさによって異なります。）
    → 画像を拡大する／縮小する
  • 「マーキー表示」
    領域指定ツールで指定した対象領域をウィンドウ全体に拡大して表示します。
    → 画像を拡大する／縮小する
  • 「1対1表示」
    画面の1ピクセルと画像データ上の1ピクセルが対応したサイズで画像を表示します。（表示される画像のサイズは、ディスプレイの解像度によって異なります。）
    → 画像を拡大する／縮小する
  • 「トーンカーブの最適化」
    トーンカーブを、領域指定ツールで指定した領域に最適化します。
    → 領域を選択する／切り抜く
  • 「トーンカーブの反転」
    表示している画像のトーンカーブの白と黒を入れ替えます。
    → 明るさ・コントラスト（トーン）を調整する
  • 「倍率をフレームに対する割合で維持」
    ビューワのフレーム内に画像全体が表示されるように自動的に倍率を調節します。
  • 「アノテーションレイヤを表示」
    画像に付けられたアノテーション（注釈）を表示します。
    → 画像にアノテーションを付ける
    → ビューワの表示設定
  • 「情報を重ねて表示」
    患者氏名、患者IDなどの情報を、画像と重ねて表示します。
    → ビューワの表示設定
  • 「スケールを表示」
    スケールを画像と重ねて表示します。
    → ビューワの表示設定
  • 「カレントビューの表示属性をコピー」
    選択した画像（カレント画像）の表示属性（移動、ズーム、トーンの情報）をコピーします。コピーされた表示属性は、「表示属性の貼り付け」によって任意の画像にペーストできます。
  • 「表示属性を貼り付け」
    「カレントビューの表示属性をコピー」でコピーした画像の表示属性をペーストします。
  • 「最初のビューにジャンプ」
    開いている最初の画像を表示して選択状態（カレント画像）にします。
  • 「最後のビューにジャンプ」
    開いている最後の画像を表示して選択状態（カレント画像）にします。
  • 「カレントビューのみ表示」
    ビューエリアにカレント画像のみ表示します。この表示を解除するには、「フレーム」メニューから、表示するフレームの枚数を指定します。
  • 「サムネイル」
    サムネイルの表示／非表示を切り替えます。
  • 「DICOM属性」
    DICOM属性の表示／非表示を切り替えます。
  • 「補間方法」
    画像補間の方法を、「ニアレストネーバー（最近傍：補間なし）」、「バイリニア（双一次）」、「バイキュービック（双三次）」から選択します。
    → ビューワの表示設定
  • 「シリーズ連動」
    シリーズ連動のオン／オフを切り替えます。オンにすると、同一シリーズの画像の移動・拡大・トーンなどの操作を同時に行います。
    → 操作する画像を選ぶ／複数の画像を同時に操作する（連動）
  • 「実寸表示の設定」
    画像を実寸で表示する際の基準の長さを調整します。実寸で表示するには、あらかじめここで調整を行ってから、インフォメーションパレットの「倍率」ドロップダウンリストで「実寸」を選択します。
    → 画像を拡大する／縮小する

「フレーム」メニュー
  • 「1x1」、「1x2」、「1x3」、「2x1」、「2x2」、「2x3」、「3x3」、「3x4」、「4x3」、「4x4」、「4x5」、「5x4」、「5x5」、「6x6」、「7x7」、「8x8」
    一度に表示する画像のフレーム数を各項目から選んで指定します。
    → ビューワの表示設定
    → ビューエリア
  • 「直接指定」
    一度に表示する画像のフレーム数を直接入力して指定します。
    → ビューワの表示設定
  • 「ビューの多表示」
    一度に表示する画像のフレーム数を、ディスク作成時のフレーム設定に戻します。
    ただし、ディスク作成時のフレーム設定が、横1行または縦1列の場合は、すべて「4x4」に設定されます。
    → ビューワの表示設定

「分析」メニュー
  • 「水平プロファイル」
    対象領域の濃淡の平均値の変化を、水平方向についてグラフ表示します。
  • 「垂直プロファイル」
    対象領域の濃淡の平均値の変化を、垂直方向についてグラフ表示します。

「ツール」メニュー
  • 「移動・ズームツール」
    ビューエリア内で画像をドラッグすることにより、画像を移動し、位置を調節します。
    また、マウスの右ボタンを押したまま、上下方向にマウスを移動することにより、画像を拡大・縮小します。
    → 画像を移動する
    → 画像を拡大する／縮小する
  • 「領域指定ツール」
    表示している画像上に対象領域（ROI）を作成します。
    → 領域を選択する／切り抜く
  • 「ズームツール」
    ビューエリア内で画像を拡大・縮小します。
  • 「トーンカーブツール」
    画像上でマウスをドラッグすることにより、トーンの変更を行います。
    マウスの左ボタンを押しながら左右に移動すると明るさ（Center値）を、上下に移動するとコントラスト（Width値）が変更されます。
    また、マウスの右ボタンを押しながら左右に移動するとγ（ガンマ）値を、上下に移動するとβ（ベータ）値が変更されます。
  • 「注釈選択ツール」
    作成した線分や注釈などのオブジェクトを選択します。
    → 画像を分析する
  • 「距離ツール」
    画像上の任意の2点間の距離を測ります。
    → 画像を分析する
  • 「曲線の長さツール」
    画像上の任意の曲線の長さを測ります。
    → 画像を分析する
  • 「3点角度ツール」
    画像上の接する2つの線分間の角度を測ります。
    → 画像を分析する
  • 「4点角度ツール」
    画像上に線分を2つ作成し、その2つの線分がなす角度を測ります。
    → 画像を分析する
  • 「多角形面積ツール」
    多角形で閉じられた領域の面積を測ります。
    → 画像を分析する
  • 「閉曲線面積ツール」
    曲線で囲まれた領域の面積を測ります。
    → 画像を分析する
  • 「心胸郭比ツール」
    心胸郭比（CTR）を測ります。
    → 画像を分析する
  • 「長さの比ツール」
    2つの線分の長さの比を測ります。
    → 画像を分析する
  • 「矢印＋文字ツール」
    注釈つきの矢印を作成します。
    → 画像にアノテーションを付ける
  • 「矢印ツール」
    注目したい場所を指示するなどのために矢印を作成します。
    → 画像にアノテーションを付ける

「リスト操作」メニュー
  • 「前の画像セットへ」
    （複数の検査画像が記録されている場合）前の診断画像を開きます。
  • 「次の画像セットへ」
    （複数の検査画像が記録されている場合）次の診断画像を開きます。

「ウィンドウ」メニュー
  • 「重ねて表示」
    今回は使用しません。
  • 「並べて表示」
    今回は使用しません。
  • 「アイコンの整列」
    今回は使用しません。
  • 「情報表示なし」
    インフォメーションパレットを非表示にします。
  • 「情報簡略表示」
    インフォメーションパレットを簡略表示します。
  • 「情報中間表示」
    インフォメーションパレットの簡略表示に加えて、倍率、長さ、濃淡などの値を表示します。
  • 「情報詳細表示」
    インフォメーションパレットを詳細表示します。
  • 「基本ツール」
    基本ツールバー（「移動・ズーム」、「トーン」、「領域指定」、「リセット」、「シリーズ」ボタン）の表示／非表示を切り替えます。（簡易版ビューワでは、「移動・ズーム」、「トーン」、「リセット」ボタンが切り替えの対象となります。）
  • 「タイムライン」
    タイムラインの表示／非表示を切り替えます。
  • 「拡張ツール」
    拡張ツールバー（「全画像選択」、「選択画像」、「全体を指定」、「領域解除」、「90°右回転」、「90°左回転」、「左右反転」、「上下反転」、「比較モード」、「切り抜き」ボタン）の表示／非表示を切り替えます。（簡易版ビューワでは、「比較モード」ボタンが切り替えの対象となります。）
  • 「オーバービューウィンドウ」
    オーバービューウィンドウの表示／非表示を切り替えます。
  • 「ステータスバー」
    ステータスバーの表示／非表示を切り替えます。
  • 「表示位置の整理」
    各画面の表示位置を調整します。
    → 画面各部の名称とはたらき

「シネモード」メニュー
  • 「シネモード」
     同一シリーズの画像、および、マルチフレーム画像をシネ表示します。
    → 画像をシネ表示する
