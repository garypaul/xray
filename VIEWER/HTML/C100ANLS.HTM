画像を分析する

ビューワで開いている画像の長さや角度を測定します。

長さを測る

2点間の距離を測る
  1. 「ツール」メニューの「距離ツール」を選択する。
  2. 測りたい距離の1点目をクリックして点を打ちます。
  3. 2点目をクリックして点を打ちます。
    手順2と手順3で打った点を線分で結んだオブジェクトが作成され、2点間 の距離が表示されます。
    距離の単位は、インフォメーションパレットの「長さ」ドロップダウンリストボックスで変更できます。

参考

表示されている数値は、「ツール」メニューの「注釈選択ツール」を選択し、ドラッグすることで、任意の位置に移動させることができます。

注意

pixel以外の単位で測るには、Pixel Spacing情報（画像を構成する、隣りあう2つのピクセルの中心間の距離）が必要です。このため、画像によっては、Pixelサイズ以外では測れないことがあります。
Pixel Spacing情報が設定されていない場合、基準となる長さを登録することにより、実測値を測ることができます。詳しくは、「計測の基準となる長さを設定する」を参照してください。

曲線の長さを測る

 画像上の任意の曲線の長さを測ることができます。
  1. 「ツール」メニューの「曲線の長さツール」を選択する。
  2. 長さを測りたい曲線の端でマウスの左ボタンを押したまま、曲線をなぞるようにマウスを移動し、なぞり終わったらボタンを離す。
    手順2でなぞった場所をつないだオブジェクトが作成され、曲線の長さが表示されます。

参考

表示されている数値は、「ツール」メニューの「注釈選択ツール」を選択し、ドラッグすることで、任意の位置に移動させることができます。

注意

pixel以外の単位で測るには、Pixel Spacing情報（画像を構成する、隣りあう2つのピクセルの中心間の距離）が必要です。このため、画像によっては、Pixelサイズ以外では測れないことがあります。
Pixel Spacing情報が設定されていない場合、基準となる長さを登録することにより、実測値を測ることができます。詳しくは、「計測の基準となる長さを設定する」を参照してください。

角度を測る

折線の間の角度を測る

 画像上の接する2つの線分間の角度を測ります。
  1. 「ツール」メニューの「3点角度ツール」を選択する。
  2. 折線の重なる交点でクリックする。
  3. 中心点から2線分のうち片方の線分の端でクリックし、一方の辺を決める。
  4. もう片方の線分の端でクリックし、もう一方の辺を決める。
    手順1でクリックした場所を中心として、手順2と手順3で作成した辺のオブジェクトが作成され、2線分間の角度が表示されます。

2線分の角度を測る

 画像上に線分を2つ作成し、その2つの線分がなす角度を測ります。
  1. 「ツール」メニューから「4点角度ツール」を選択する。
  2. 2線分のうちの片方の線分の端をクリックして点を打つ。
  3. 反対側の端をクリックして点を打つ
    手順2と手順3で決めた点をつなぐ線分が表示されます。
  4. もう片方の線分の端をクリックして点を打つ。
  5. その反対側の端点をクリックして点を打つ。
    手順4と手順5で決めた点をつなぐ線分が表示されます。
    作成された2本の線分の延長線が交わったときの角度が表示されます。

面積を測る

多角形の面積を測る

 多角形で閉じられた領域の面積を測ります。
  1. 「ツール」メニューの「多角形面積ツール」を選択する。
  2. 面積を測りたい多角形の角を順次クリックする。
    クリックした箇所に点が打たれ、多角形の角になります。点を増やしていくと、点と点を結ぶ線分が作られます。
  3. 最後の接点で、2回クリックする。
    最初の角と最後の角が線分で結ばれ、多角形のオブジェクトが作成されて面積が表示されます。

曲線で囲まれた面積を測る

 曲線で囲まれた領域の面積を測ります。
  1. 「ツール」メニューの「閉曲線面積ツール」を選択する。
  2. 測りたい領域をドラッグしてなぞり、終わったらボタンを離す。
    最初にマウスボタンを押した場所と、最後にボタンを離した場所が線分で結ばれ、閉じた曲線のオブジェクトが作成されて面積が表示されます。

長さの比を測る

 2線分間の比や心胸郭比（CTR）を測ります。

2線分の比を測る

 2つの線分の長さの比を測るには、以下の操作を行います。
  1. 「ツール」メニューの「長さの比ツール」を選択する。
  2. 比較元の線分の端をクリックして点を打つ。
  3. 反対側の端をクリックして点を打つ。
    手順2と手順3で決めた点が線分で結ばれます。
  4. 比較するもう1つの線分の片方の端をクリックして点を打つ。
  5. 反対側の端をクリックして点を打つ。
    手順4と手順5で決めた端点が線分で結ばれます。
    作成された2本の線分の長さの比が表示されます。

心胸郭比（CTR）を測る

 心胸郭比（CTR）を測るには、以下の操作を行います。
  1. 「ツール」メニューの「心胸郭比ツール」を選択する。
  2. 背骨の上部と下部の端をそれぞれクリックして背骨の位置を決める。
    指定した点が線分で結ばれます。
  3. 胸郭の両端をそれぞれクリックして胸郭の幅を決める。
  4. 心臓の両端をそれぞれクリックして心臓の幅を決める。
    心胸郭比（CTR）が表示されます。
